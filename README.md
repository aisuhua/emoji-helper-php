## 项目简介

剔除、清除、替换、判断、输出emoji表情的一个php类，对没有使用utf8mb4编码数据库的项目，你懂的，这个少不了。

## 提供了以下功能：

*  1. 判断字符串是否包含emoji表情
*  2. 清除字符串中的emoji表情
*  3. 将emoji表情的16进制输出为字符串 (用于补全表情数组)
